# isolation_forest

## Isolation Forest 이란

Isolation Forest는 정상 데이터는 서로 비슷해서 분리하기 어렵고,
이상치는 주변과 동떨어져 있어서 빨리 분리된다는 아이디어를 쓰는 방법임.

즉,

정상값 → 트리에서 여러 번 쪼개야 고립됨

이상값 → 몇 번 안 쪼개도 바로 고립됨

그래서 평균 경로 길이(path length) 가 짧은 점을 이상치로 판단함.

## 전체 흐름

이 튜토리얼의 흐름은 아래와 같음.

1. 데이터 준비
2. Isolation Forest 모델 생성
3. 모델 학습
4. 이상치 예측
5. 이상치만 따로 뽑기
6. 시각화로 확인

그리고 방법을 두 가지로 보여줌.

방법 1: contamination 값을 직접 주고 fit_predict() 사용

방법 2: score_samples()로 점수를 얻고, 직접 하위 몇 %를 임계값으로 잘라서 이상치 판별
